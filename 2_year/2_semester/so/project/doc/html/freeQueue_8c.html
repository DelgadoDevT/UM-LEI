Detailed Description

Provides functions to manage a free queue, which persists to disk.

This module provides operations for initializing the free queue, adding and removing items, saving the queue to disk, and clearing the queue. The free queue is used to manage reusable values (e.g., memory blocks) that are stored persistently between sessions.
