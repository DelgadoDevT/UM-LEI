# Human-Computer Interaction Course
This folder contains the work developed during the Human-Computer Interaction (Interface Pessoa-Máquina) course at Universidade do Minho.

## Overview
The course explores the principles of user-centered design, prototyping, and usability. Practical assignments include building responsive web interfaces, implementing interactive applications (like a To-Do app and a Connect Four game), and developing a comprehensive final project focused on dashboard design and data visualization.

## Tech Stack
* **Frontend:** Vue.js 3, Vite, HTML5, CSS3 (Media Queries), JavaScript (ES6+)
* **Tools & Concepts:** Prototyping (Figma/Relatório), JSON Server (Mock API), UI/UX Design Principles, Responsive Design.

## Acknowledgements
Several practical exercises and base project templates were provided by the teaching staff. The implementation of the frontend logic, component architecture, and interface styling was conducted as part of the course evaluation.
